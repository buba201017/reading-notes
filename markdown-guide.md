# Aprendiendo Markdown

## Encabezados de diferentes niveles

Para crear encabezados de diferentes niveles, se usan los símbolos de almohadilla (`#`). El número de almohadillas indica el nivel del encabezado.

```markdown
# Título de nivel 1
## Título de nivel 2
### Título de nivel 3
#### Título de nivel 4


## Texto en negrita e itálica

Puedes utilizar asteriscos (`*`) o guiones bajos (`_`) para dar formato al texto.

**Negrita**: Usa dos asteriscos (`**`) o guiones bajos (`__`) para poner el texto en negrita.

**Itálica**: Usa un solo asterisco (`*`) o guión bajo (`_`) para poner el texto en itálica.

## Listas ordenadas y no ordenadas

- Listas no ordenadas se crean con un guion (`-`), un asterisco (`*`), o un más (`+`).
. Elemento 1
. Elemento 2
. Elemento 3


- Listas ordenadas se crean con números seguidos de un punto (`1.`, `2.`, `3.`).
1.
2.
3.


## Enlaces a recursos útiles

Puedes agregar enlaces utilizando corchetes (`[]`) seguidos de paréntesis (`()`).


## Tabla comparativa entre HTML y Markdown

Las tablas en Markdown se crean usando guiones (`-`) y barras verticales (`|`) para separar las columnas.

| Característica     | HTML                                      | Markdown                                |
|--------------------|-------------------------------------------|-----------------------------------------|
| Encabezados        | `<h1>`, `<h2>`, `<h3>`...                 | `#`, `##`, `###`...                     |
| Negrita            | `<strong>Texto en negrita</strong>`       | `**Texto en negrita**`                  |
| Itálica            | `<em>Texto en itálica</em>`               | `*Texto en itálica*`                    |
| Enlaces            | `<a href="url">Enlace</a>`                | `[Enlace](url)`                         |
| Listas ordenadas   | `<ol><li>Elemento 1</li></ol>`            | `1. Elemento 1`                         |
| Listas no ordenadas| `<ul><li>Elemento 1</li></ul>`            | `- Elemento 1`                          |
